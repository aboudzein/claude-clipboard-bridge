# Changelog

## [1.0.0] - 2026-03-04

### Added
- System tray app with status icons (blue/green/gray)
- Context-aware clipboard swapping (terminal = path, other apps = image)
- Efficient clipboard polling via `GetClipboardSequenceNumber`
- SHA-256 image deduplication
- Atomic file writes (temp + rename)
- Image history with configurable cleanup
- Balloon notifications on screenshot save
- Configurable terminal process list via `config.json`
- One-liner remote installer
- `.bat` launchers for double-click install/uninstall
- Auto-start via Windows Task Scheduler
- Start Menu shortcut
- Pause/Resume via tray icon
